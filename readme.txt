== Description ==

This is a Cheetahpay Payment Gateway for WooCommerce.

Cheetahpay primarily converts your airtime to cash, with this plugin, you can securely sell your products and subscriptions online while receiving airtime as payment.

Integrate this plugin into your woocommerce website is very simple and our support system is always there to guide you via call, SMS or Whatsapp 07062453689.

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

If on the off-chance you do encounter issues with the shop/category pages after an update you simply need to flush the permalinks by going to WordPress > Settings > Permalinks and hitting 'save'. That should return things to normal.

== Frequently Asked Questions ==

= Does this plugin work with any Telecom Network? =

Cheetahpay currently supports MTN, AIRTEL and 9 MOBILE Nigerian networks 

= Does this support Checkout with Cheetahpay from the cart view? =

Yes!

= How do I configure this plugin? =

To use this plugin, You need to create an account with cheetahpay and follow the instructions on Wordpress/woocommerce integrtion [user guide](http://cheetahpay.com.ng/#developers)

= Where can I get support or talk to other users? =

If you get stuck, you can ask for help in the Plugin Forum.

Yes, this plugin will work with any theme, but may require some styling to make it match nicely. If you're
